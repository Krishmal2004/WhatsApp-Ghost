# WhatsApp-Ghost

Send WhatsApp messages where the receiver can see the message, but the sender identity is not shown (or is obscured), depending on the method used.

> Disclaimer: “Ghost/anonymous” WhatsApp messaging can easily be used to harass, spam, impersonate, or violate privacy. Use this project only for legitimate, ethical purposes and in compliance with WhatsApp’s Terms of Service and local laws. I’m not responsible for misuse.
